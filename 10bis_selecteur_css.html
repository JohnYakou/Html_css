Propriété CSS utilisé dans cette page

Décrivez leurs usages et les valeurs possibles en fonction de ce que vous avez déjà vu

  1. background : modifie l'arrière-plan (couleur, taille), il peut prendre plusieurs valeurs (reappeat, url( ), fixed)
  2. color : permet de modifier la couleur du texte
  3. float : indique qu'un élément doit être retiré du flux normal (= de haut en bas) et doit être placé sur le côté droit ou sur le côté gauche de son conteneur
  4. font-family : définit la police d'un texte (times new roman)
  5. font-size : définit la taille de fonte utilisée pour le texte (px, %, em, rem)
  6. height : hauteur qui se modifie (px, %, vh, vw)
  7. left : permet de positionner à gauche (px), ça fonctionne avec une propriété position
  8. line-height : espacement entre les lignes de texte (px, %, em, rem)
  9. margin : définit la taille des marges sur les quatre côtés de l'élément, peut prendre jusqu'à 4 valeurs (top, right, bottom, left) (px, %, em, rem)
  10. padding : permet de définir les différents écarts de remplissage sur les quatre côtés d'un élément. Elle synthétise padding-top, padding-right, padding-bottom, padding-left.
  11. position : définit la façon dont un élément est positionné dans un document (top, right, bottom et left) (fixed, stastic, relative, ...)
  12. text-align : aligne le texte (center, justify, right, left)
  13. text-decoration : enlève le soulignement avec la valeur none
  14. top : fonctionne avec position, définit une partie de la position des éléments positionnés, elle n'a aucun effet pour les éléments non-positionnés
  15. transition : précise un paramètre d'une animation comme sa durée, peut prendre une valeur en secondes
  16. widht : permert de modifier la largeur d'une balise (px, %, vh, vw)

Propriété CSS à rechercher

Rechercher les propriétés ci-dessous, faites des tests avec cette page depuis la console de debuggage (F12 sur Firefox ou clic droit "inspecter")

  1. font-weight : permet de définir la graisse utilisée pour le texte (bold, 1 à 1000)
  2. letter-spacing : définit l'espace entre chaques caractères (px, em, rem)
  3. word-spacing : définit l'espace entre les mots (px, em, rem)
  4. text-shadow : ajoute des ombres au texte (1px, 1px, 10px, couleur)
  5. background-image : ajoute une ou plusieurs images comme arrière(s)-plan(s) (url(), un dégradé comme valeur)
  6. background-size : définit la taille du background (cover, px, %, contain)
  7. opacity : définit la transparence d'une balise (de 0 à 1)
  8. border-style : propriété qui permet de définir le style des lignes utilisées pour les bordures des quatre côtés de la boîte d'un élément (dotted, dashed, solid, double, groove, ridged)
  9. box-shadow : ajoute des ombres derrière une balise (px, % - x, y, blur, diffusion)
  10. list-style-image : remplace la puce d'une liste par une image (avec un url)
  11. border : ajoute et définit les propriétés liées à la bordure ; sa taille (en px), ses formes (solid, dashed, dotted, ...) et ses propriétés (color, widht, height)
  12. border-radius : arrondis les coins de la bordure (px, %, em, rem)
  13. border-width : définit la largeur de la bordure d'un élément (px, emn rem)
  14. font-variant : modifie le style de la police (plusieurs valeurs [small-caps, normal]
  15. background-color : permet de définit la couleur de l'arrière-plan, le background
  16. !important : utilisé à la fin d'une ligne pour ajouter plus d'importance à une propriété/valeur que la normale
